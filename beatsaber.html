BeatSaber modded level

This assignment was great for me because BeatSaber is one of my favorite games. I had made my own level once before but not with code. I had first asked for help in a Discord where people could help me get started. I first went and read through a GitHub page that helped me link code to my level which allowed me to display my blender files in the level. I could also move walls, bombs, blocks, lazers, camera movements, with code giving me very many options to make it beautiful. The assignment for school to divide it into 5 different parts so I also used different code and different blender files. I am very proud of this assignment and hope to do it another time.
