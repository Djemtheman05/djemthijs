BeatSaber modded level


Deze opdracht was geweldig voor mij omdat BeatSaber een van mijn favoriete spellen is. Ik had al eens een eigen level gemaakt eerder gemaakt maar niet met code. Ik had eerst hulp gevraagd in een Discord waar mensen me op weg konden helpen. I ging eerst een GitHub pagina doorlezen die me hielp om code aan mijn level te koppelen waardoor ik mijn blender bestanden in het level kon weergeven. Ik kon ook muren, bommen, blokken, lazers, camerabewegingen, met code verplaatsen waardoor me heel veel opties om het mooi te maken. De opdracht voor school om het op te delen in 5 verschillende delen dus ik ook verschillende code en verschillende blender bestanden gebruikt. Ik ben erg trots op deze opdracht en hoop het nog eens te doen een andere keer.
