VR goalkeeper game

I personally had a lot of fun with this assignment because I absolutely love virtual reality, no matter if I'm playing it or enjoying the environment. Making the goalie game was especially fun because I could see my own environment and objects in a virtual world, and of course play it on my VR glasses. Starting the project was difficult though because you have to specify that you want to use your vr glasses and controllers for this project and also make them function naturally. Once that was done I was able to continue making code work, and movement in the game until I had a functioning end result. If you stop 10 balls you have won. But if the opponent scores 10 you have lost. This is still one of my favorites projects.
